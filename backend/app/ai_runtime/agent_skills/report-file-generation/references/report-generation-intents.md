# Report Generation Intent Reference

Trigger examples:

- 生成报告
- 导出报告
- 生成文件
- 下载报告
- 保存为 Word
- 保存为 DOCX
- 保存为 PDF
- 把刚才回答整理成报告
- 把本轮分析做成文件
- create a report file
- export as docx
- download as pdf

Non-trigger examples:

- 报告里写了什么
- 总结报告内容
- 分析上传报告
- 报告碳核算怎么算

These non-trigger examples may need RAG/file reading/carbon calculation, but not file export.
