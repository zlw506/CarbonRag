## MODIFIED Requirements

### Requirement: Frontend exposes a knowledge base workbench

CarbonRag SHALL provide a knowledge base workbench for document status, chunk preview, test QA, and RAG trace inspection.

#### Scenario: User validates retrieval before AskPage use

- **WHEN** a user runs test QA from the workbench
- **THEN** the UI shows retrieved chunks, scores, backend status, and citations.
