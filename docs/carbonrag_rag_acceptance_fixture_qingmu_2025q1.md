如采用 market-based 口径，应保留凭证编号、交易月份、覆盖电量和适用设施。

## 4. 排放因子口径

外购电力 location-based 核算应优先使用对应地区或省级电力平均排放因子。
若无法匹配省级因子，可回退到区域因子。
若区域因子也不可用，才回退全国平均因子。

使用回退因子时，系统必须在 warnings 中说明回退原因。

燃料燃烧因子若来自临时 guidance_default 或 demo 因子，不得标记为 official。
正式企业核算应优先使用国家温室气体排放因子数据库或适用行业指南。
