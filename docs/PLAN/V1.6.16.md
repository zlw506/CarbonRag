# V1.6.16 Local Open-Source LLM Runtime

## Summary

- V1.6.16 keeps RAG-Pro as the RAG implementation spine and switches CarbonRag's default chat generation route toward local open-source LLMs.
- RAG-Pro does not ship a DeepSeek chat model package; it configures chat providers. CarbonRag follows that pattern by using local OpenAI-compatible endpoints such as Ollama, vLLM, or LM Studio.
- BGE-M3 and bge-reranker remain retrieval/rerank models, not chat models.
- 当前 DeepSeek 聊天模型包和 Ollama runtime 尚未准备好；项目负责人仍在研判后续是否采用嵌入式模型加载方式。本轮提交的是 local LLM profile、smoke、文档和边界，不宣称本地聊天模型已可用。

## Key Changes

- `.env.example` now defaults to local OpenAI-compatible chat at `http://127.0.0.1:11434/v1` with `MODEL_NAME=deepseek-r1:8b`.
- Added local LLM runtime profiles:
  - `.env.llm.ollama-deepseek.example`
  - `.env.llm.vllm-deepseek.example`
  - `.env.llm.lmstudio.example`
- Added smoke scripts:
  - `scripts/llm-smoke-openai-compatible.ps1`
  - `scripts/llm-smoke-openai-compatible.sh`
- Added `backend/local_provider_config.ollama-deepseek.example.json` for users who prefer the existing local provider override file.
- Documented offline chat model package placement under `data/outputs/models/LLM/<model-name>/`.

## Operational Guidance

Recommended first route:

```powershell
# Start Ollama separately, then run:
$env:MODEL_API_BASE_URL="http://127.0.0.1:11434/v1"
$env:MODEL_API_KEY="ollama-local-key"
$env:MODEL_NAME="deepseek-r1:8b"
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/llm-smoke-openai-compatible.ps1
```

If the model package is offline, ask #1 for it and place it under:

```text
data/outputs/models/LLM/deepseek-r1-8b/
```

## Boundaries

- Do not commit local model weights.
- Do not confuse chat LLM with embedding model.
- Do not replace BGE-M3 embedding with a normal chat-completions API.
- Keep RAG-Pro direct port/rewrite permission active for RAG development.

## Verification

- `scripts/llm-smoke-openai-compatible.ps1` must return a non-empty chat answer.
- KnowledgeBaseWorkbench Test QA must be able to call the active chat provider.
- AskPage with selected KB must return a grounded answer, citations, and retrieval trace.
- 在 DeepSeek/Ollama/嵌入式 runtime 未确定前，V1.6.16 验收只确认配置与 smoke 工具可用；真实模型连通性等待项目负责人确认 runtime 路线后再跑。
