# V1.5.11 会话标题时机与会话控制

## 目标

- 第一次用户发送后立即生成临时会话标题。
- 第二次用户发送后再修正标题。
- 第三次及以后 AI 不再自动改标题。
- 会话栏支持重命名、置顶、删除。

## 边界

- 不改 #2 RAG 主线。
- 不改 #3 聊天折叠工作。
- 不新增复杂项目/归档/分享能力。

## 验收

- `PATCH /api/v1/sessions/{id}` 可更新标题和置顶状态。
- `DELETE /api/v1/sessions/{id}` 可删除当前用户自己的会话。
- 流式首包可携带标题更新状态，前端会话栏即时刷新。
- 前端会话栏三点菜单可操作重命名、置顶、删除。
