# CarbonRag

当前状态：第 0 轮，项目生产规范与文档基线冻结完成。

## 项目定位
CarbonRag 是一个面向中小企业的“双碳”政策与企业应用智能问答 MVP，目标是围绕政策理解、企业样例接入、基础碳核算和报告生成形成首条可演示闭环。

## 当前阶段
- 阶段名称：第 0 轮文档冻结轮
- 当前策略：统一使用第三方云端 API，保留后续替换为本地或私有模型的接口能力
- 当前原则：先冻结规则与边界，再进入代码骨架施工

## 第 0 轮目标
- 冻结项目定位、团队职责、技术栈基线、Git 工作流和 MVP 范围
- 建立仓库目录骨架、环境变量模板和忽略规则
- 完成首轮本地 Git 初始化与基线提交

## 当前阶段不做
- 本地开源大模型正式部署
- 微调链路与量化部署
- 复杂权限系统与多租户体系
- 全国政策全量自动更新
- 生产级监控、审计与复杂流程编排
