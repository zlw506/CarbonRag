# public 目录说明

本目录只允许放入公开可引用的政策、标准、指南和说明性样本文档。

## 当前用途

- 为 `ask mode` 提供首条公共政策 grounding 样本语料
- 支撑 `policy_retrieve` 的本地轻量检索
- 为前端 citations 展示提供可追溯来源

## 目录结构

```text
data/public/
├─ corpus/
│  ├─ docs/
│  └─ manifest.json
└─ README.md
```

## 允许内容

- 国家级双碳政策样本
- 地方政策样本
- 公开标准与公开指南
- 基于官方公开来源整理的标准化摘录文本

## 禁止内容

- 受限传播资料
- 真实企业内部文档
- 无法确认公开授权的数据
- 自动抓取但未复核来源的文本

## v0.1.6 当前语料口径

- 当前只建立最小公共政策样本语料，不引入 PDF 原文与自动爬虫
- `corpus/docs/*.md` 采用人工整理摘录，保留 `doc_id`、标题、发布机构、来源链接、日期、地域和类型元信息
- `manifest.json` 是 loader 的统一索引入口
- 当前 citations 仅来自本地公共政策样本，不代表已完成完整 RAG 平台建设
